# Contributing

Thanks for helping improve this project.

## Local setup

1. Use Node.js `24` (the repo also includes `.nvmrc`).
2. Install dependencies with `npm ci`.
3. Run the full verification flow with `npm run verify`.

## Pull requests

1. Keep changes focused and easy to review.
2. Add or update tests when behavior changes.
3. Make sure `npm run verify` passes before opening or updating a pull request.
