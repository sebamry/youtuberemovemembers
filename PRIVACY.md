# Privacy Policy

Last updated: 2026-03-17

## Overview

YouTube Members Filter is a Chrome extension that hides members-only YouTube content on supported YouTube pages.

The extension is designed to work locally on the user's device.

## Data Collection

YouTube Members Filter does not sell personal data.

YouTube Members Filter does not transfer user data to third parties.

YouTube Members Filter does not use remote code.

YouTube Members Filter does not collect or send personal information, browsing history, authentication tokens, or payment information to any server.

## Data Stored Locally

The extension stores the following data locally in Chrome storage on the user's device:

- whether the filter is enabled
- per-surface toggle preferences
- theme preference
- whitelisted channel identifiers
- a local history of hidden video IDs used for the hidden-content counter

This data stays in the user's browser profile unless the user removes it or uninstalls the extension.

## Permissions

The extension uses:

- `storage`: to save settings, theme preference, whitelist entries, and local counters
- `https://www.youtube.com/*`: to detect and hide members-only content on YouTube pages

## No Remote Processing

YouTube Members Filter performs its filtering logic locally in the browser. It does not upload page contents or account information for processing.

## Changes

If this privacy policy changes, the updated version should be published at the same public URL used in the Chrome Web Store listing.
