# Chrome Web Store Assets

This directory contains ready-to-upload store assets for the Chrome Web Store listing.

## Included Files

- `store-screenshot-overview.png` - 1280x800 overview shot with before/after filtering
- `store-screenshot-settings.png` - 1280x800 settings shot covering theme, whitelist, and stats
- `store-small-promo-tile.png` - 440x280 promo tile

## Source Files

Editable source layouts live in `./source/`.

Use them if you want to regenerate updated PNGs before a future store refresh.
